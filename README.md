# MaskCara

Juego realizado por:
Aaron Rumbos (Artista)

Miguel Orsini (Artista)

Raul Diaz (Programador)

Edgar Sakellariou (Programador)


Juego realizado en la Global Game Jam 2026, sede Maracay, Venezuela.

Puedes jugarlo en: https://cccoriolis.itch.io/maskcara

Crea mascaras artesanales cumpliendole los deseos a tus clientes, sigue el manual de dioses para no equivocarte con sus peticiones.
---

## Tecnologías utilizadas

- Unity
- C#
- Visual Studio
- Blender
- Photopea
---

## Características principales

- State Machine Basico
- Sistema de aleatoriedad de clientes y sus peticiones.
- Diseño de interfaz dinamico

---

## Arquitectura del proyecto

El proyecto está organizado utilizando principios de programación orientada a objetos y separación de responsabilidades.

Ejemplo de módulos:

GameManager  
CustomerController  
MaskAssetSystem
CustomerIA
UIManager
